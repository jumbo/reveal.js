环境

  • Mac
    • # Bash
$ gem install sass
$ gem install compass
  • 正常PC
    安装 Ruby
    # CMD
$ gem install sass
$ gem install compass
  • 异常PC 
    # CMD
# 从rubygems.org下载相应的gems
# (rake,rspec,diff-lcs,fssm)等
$ gem install ./sass
$ gem install ./$gems
$ gem install ./compass

环境

  • 检查是否安装成功 
    # Bash
$ sass -v
$ Sass 3.2.10 (Media Mark)
$ compass -v
$ Compass 0.12.2 (Alnilam)
  • 更换ruby的安装源 
    # Bash
# 使用淘宝提供的镜像服务器
$ gem sources --remove https://rubygems.org/
$ gem sources -a http://ruby.taobao.org/
$ gem sources -l
*** CURRENT SOURCES ***

http://ruby.taobao.org
# 确保只有 ruby.taobao.org
  • 环境变量$PATH

常用命令

  • sass
    • watch 
      # Bash
$ sass input.scss output.css
$ sass --watch input.scss:output.css
$ sass --watch app/sass:public/stylesheets
    • sass-convert 
      sass-convert －F css -T scss input.css output.scss
  • compass
    • create/init 
      # Bash demo
$ compass create path/to/project
    • watch 
      $ cd path/to/project
$ compass watch

新建项目

  • compass 
    # Bash
$ mkdir compass-test
$ cd compass-test
$ compass init
$ compass watch
