环境

  • Mac
    • # Bash
$ gem install sass

  • 正常PC
    安装 Ruby
    # CMD
$ gem install sass
$ gem install compass
  • 异常PC 
    # CMD
# 从rubygems.org下载相应的gems
# (rake,rspec,diff-lcs,fssm)等
$ gem install ./sass
$ gem install ./$gems
$ gem install ./compass

附加

  • 检查是否安装成功 
    # Bash
$ sass -v
$ Sass 3.2.10 (Media Mark)
$ compass -v
$ Compass 0.12.2 (Alnilam)
  • 更换ruby的安装源 
    # Bash
$ gem sources --remove https://rubygems.org/
$ gem sources -a http://ruby.taobao.org/
$ gem sources -l
*** CURRENT SOURCES ***

http://ruby.taobao.org
# 请确保只有 ruby.taobao.org

试用

  • sass试用

  • compass试用 
    # Bash
$ mkdir compass-test
$ cd compass-test

$ compass watch

基础命令

  • sass
    • watch/complie 
      # Bash
$ sass --watch input.scss:output.css
$ sass --watch app/sass:public/stylesheets
    • sass-convert 
      sass-convert input.css output.scss -T scss
  • compass
    • create 
      # Bash demo
$ compass create path/to/project
    • init 
      $ cd path/to/project
$ compass init
